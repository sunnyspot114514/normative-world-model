- **Model revision/weight validation:** the served path is pinned to the exact revision snapshot; validation checks config + index, exactly 21 shard names from the index, hardcoded byte total, and records per-shard SHA-256 (`runner-used.sh:60-99`). `/v1/models` `root` confirms the exact snapshot was served (`client-result.json:21`). All four must nevertheless be treated as frozen inputs for any future comparison (see §6).
