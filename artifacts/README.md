# Artifacts

Final tables, plots, reports, audit bundles, and exportable manifests belong here. Large or
regenerable artifacts remain untracked; only intentionally curated small summaries should be
considered for version control.

The revision-2 smoke generator writes its exit report, dataset card, provenance manifest,
uncertainty reachability table, and external-audit bundle under `phase1_revision2_smoke/`.
Retained Phase-1 artifacts use `phase1_v2/` only after the external acceptance lock is satisfied.
